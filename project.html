Final Project

For the final project in the APCS course, students had to write Java code implementing inheritance with student-designed classes and interfaces. I created a program simulating a candy store in which users can fill up a virtual shopping cart and checkout. Users can choose between 6 types of candy and specify the amount they wish to purchase. There is also an option to fill the cart with a random assortment.

Candy

- A class that holds information about a type of candy. This includes the price and number of calories in 100g of this candy.

Order

- A class that holds information about one of the user's orders. Each order includes the type of candy, amount ordered in grams, and total price for this order.

Shoppable

- An interface for shoppable objects. Shoppable objects allow the user to add items to their virtual shopping cart, calculate the total price of their cart, and print out the contents of the cart.

Store

- A class for any type of store. Users can open and close the store, get information about its size in square feet, change the time, and manage a list of ongoing discounts.

CandyStore

- A class for a candy store. Users have the option of manually creating orders, each time specifying candy type and amount, or allowing the program to randomly fill the cart for them. If they choose to randomly fill, users can even specify the types of candy from which the store can randomly choose.

CandyStoreDriver

- A driver class to run the program. The driver class creates one candy store, "Sweet Factory," that allows users to manually select candy. The class also creates another store, "Candyland," which allows users to randomly fill their cart.
